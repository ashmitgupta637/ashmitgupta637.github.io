# Terminal-Themed Personal Portfolio

A responsive, terminal-themed personal portfolio website to showcase software and hardware engineering projects, skills, and achievements[cite: 1].

## Live Demo

[https://ashmitgupta637.github.io/](https://www.google.com/search?q=https://ashmitgupta637.github.io/)

## Features

* **Interactive Command-Line Console:** Built-in terminal simulator supporting commands like `help`, `whoami`, `projects`, `hardware`, and `clear`[cite: 1].
* **Live GitHub Stats:** Fetches and displays real-time GitHub statistics (repos, followers, stars, top languages) using the GitHub REST API[cite: 1].
* **Animated UI Elements:** Includes CRT scanline overlays, typing animations, interactive progress meters, and SVG hardware circuit animations[cite: 1].
* **Responsive Design:** Fully optimized for both desktop and mobile viewing[cite: 1].

## Technologies Used

* **HTML5:** Semantic structure and layout[cite: 1].
* **CSS3:** Custom properties, grid/flexbox layouts, keyframe animations, and terminal aesthetics[cite: 1].
* **JavaScript (Vanilla):** DOM manipulation, interactive console logic, Intersection Observer for animations, and asynchronous API fetching[cite: 1].
* **REST APIs:** GitHub API integration[cite: 1].

## Local Setup

1. Clone the repository:
```bash
git clone https://github.com/ashmitgupta637/ashmitgupta637.github.io.git

```


2. Navigate to the project directory:
```bash
cd ashmitgupta637.github.io

```


3. Open `index.html` in your web browser.
