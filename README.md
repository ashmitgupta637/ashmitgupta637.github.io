# Terminal-Themed Personal Portfolio

A responsive, terminal-themed personal portfolio website to showcase software and hardware engineering projects, skills, and achievements.

## Live Demo

https://ashmitgupta637.github.io/

## Features

* **Interactive Command-Line Console:** Built-in terminal simulator supporting commands like `help`, `whoami`, `projects`, `hardware`, and `clear`.
* **Live GitHub Stats:** Fetches and displays real-time GitHub statistics (repos, followers, stars, top languages) using the GitHub REST API.
* **Animated UI Elements:** Includes CRT scanline overlays, typing animations, interactive progress meters, and SVG hardware circuit animations.
* **Responsive Design:** Fully optimized for both desktop and mobile viewing.

## Technologies Used

* **HTML5:** Semantic structure and layout.
* **CSS3:** Custom properties, grid/flexbox layouts, keyframe animations, and terminal aesthetics.
* **JavaScript (Vanilla):** DOM manipulation, interactive console logic, Intersection Observer for animations, and asynchronous API fetching.
* **REST APIs:** GitHub API integration.

## Local Setup

1. Clone the repository:

   ```
   git clone https://github.com/ashmitgupta637/ashmitgupta637.github.io.git
   ```

2. Navigate to the project directory:

   ```
   cd ashmitgupta637.github.io
   ```

3. Open `index.html` in your web browser.
